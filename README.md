# 🎥 抖音无水印解析下载器 (Douyin-Downloader)

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Nuitka](https://img.shields.io/badge/Compiled_with-Nuitka-orange.svg)

一款基于 Python 编写的现代化 UI 抖音视频无水印解析工具。支持粘贴分享链接一键解析，自动捕获无水印视频流并提供本地下载功能。



## ✨ 功能特性

* **极简 UI**: 采用 `CustomTkinter` 打造的抖音风格深色模式界面，极简且专业。
* **智能识别**: 自动从复杂的分享文案中提取短链接，并根据视频标题自动命名文件。
* **一键解析**: 利用 `Playwright` 自动化引擎实时捕获视频流，突破传统 API 限制。
* **批量操作**: 支持一键复制无水印直链或直接下载视频到本地。
* **绿色单文件/文件夹**: 支持 Nuitka 编译，内置浏览器内核，无需环境即可运行。

## 🚀 快速开始

### 方式 A：直接运行 (EXE 版本)
1. 从 [Releases](../../releases) 页面下载最新的压缩包。
2. 解压后运行 `无水印视频解析.exe`。
